JAG - Java Genetic Algorithm Processor

JAG is a program and a library for the rapid development of Genetic Algorithms in the Java programming language. It does not relieve you of the knowledge necessary to write a good fitness function or crossover, but it does allow you to focus on those issues and not on the plumbing of a good GA. Please follow the directions below to install JAG to your computer.

JAG was last modified April 21st, 2005 (12:05 PM). Current version is 1.2.0

Using the link below you should be able to install JAG - Java Genetic Algorithm Processor to your personal machine. Once downloaded and installed, JAG can work whether your computer is connected to the internet or not. If your computer is connected to the internet when you start JAG, it will check this website to see if there is an updated version of the software and automatically install it, so that you do not have to worry about keeping the software up to date.

JAG will ask for all permissions on your machine. Mainly this is for write permissions to store generated pictures of CA if you use it for such.

NOTE - JAG is approximately 600 Kilobytes in size. It may take some time to download to your computer if you are using a slow (for instance, dialup) connection.

JAG may pop up BEHIND this window after installing. Once you have run the program, please check your task bar to see if JAG is now running. If so, minimize this window to see it!

Install 'JAG - Java Genetic Algorithms' to my computer

Before you can install JAG, your computer must have a Java Runtime Environment. We have detected that you do not have this software installed. To install it, please click the link below, and when your browser asks, select that you wish to OPEN the file (not SAVE it). After you have finished with that process, return to this page and hit "REFRESH" on your browser, and you should have new instructions on how to finish installing Numbers.

NOTE - The JRE is approximately 15 Megabytes. It may take some time to download. Also, the JRE listed here is for Windows machines only. Please contact me if you need help with other machines. You may have to restart your machine and return to this page after the JRE is installed to finish installing JAG.

Install the JRE on my Windows computer

Why install the JRE?

Java is a special programming language that allows software to be written once, and then run on various platforms (such as Windows, Linux, and Macintosh machines). All Lexikos software is written in Java, so once you have the JRE installed, you will be able to run any future Lexikos tools developed here without having to download it again!
